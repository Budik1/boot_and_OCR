git add .
git commit -m ""
git push

git pull - обновить все файлы

git branch - показывает доступные ветки на локальном репозитории
git branch имя ветки - создать новую ветку с именем "имя ветки"
git branch -d имя ветки - удалить. При удалении Git проверит - произошло ли полное слияние
git branch -D имя ветки - удалить без проверки

git checkout имя ветки - переключится на ветку "имя ветки"
git checkout -b имя ветки - создать и переключится
